# Secure System Administration Tutorial

This tutorial covers foundational system administration practices for securing Kali-based systems.

## Learning outcomes

- Harden user accounts and authentication.
- Configure firewalls and secure services.
- Understand logging and update strategies.

## Walkthrough

1. **Account hygiene**
   - Create a non-root user.
   - Enforce strong password policies.
2. **Service hardening**
   - Disable unnecessary services.
   - Harden SSH (disable root login, change default port, use keys).
3. **Firewall basics**
   - Enable UFW or nftables rules.
   - Restrict inbound traffic to lab requirements.
4. **Updates**
   - Configure unattended upgrades or weekly update schedules.

## Operational tips

- Snapshot before major changes.
- Keep documentation of all modifications.
- Use Clawdbot for checklists and reminders.
